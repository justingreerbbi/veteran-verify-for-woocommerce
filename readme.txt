== Description ==

**Give back by providing military discounts to your customers. This plugin will require customers that want to use a military discount to verify they are an actual Veteran or Active Service Member of the U.S. Armed Services before being allowed to checkout with a military discount code applied.**

The verification process is seamless and uses [Veteran Verify Services](https://veteranverify.app) to ensure that customer trying to use a military discount **verifies** their military status. This allows for your WooCommerce store to offer discounts only to Military Personnel.

This plugin does require at least a free account at:
[Veteran Verify Services](https://veteranverify.app)

Use your API Key from your account at [https://veteranverify.app](https://veteranverify.app) to integrate the verification services.

= Plugin Features =
* Make any coupon code in WooCommerce a Military Discount Code.
* This plugin will automatically check if a customer has applied a military discount code but has not verified yet.
* Seamless redirection between the verification services and your WooCommerce store.

== Frequently Asked Questions ==

= Is this plugin free to use? =
Yes, but it does require an an API Key from [https://veteranverify.app](https://veteranverifiy.app)

= Are there usage limitations? =
There is no limitation built into this plugin. There is a limit on the verification API which is 10 successful verifications a month. All upgraded accounts have unlimited API requests.

The API does have a rate limit which is 1 request every 30 seconds for free users and unlimited for upgraded plans.

= What happens if the API limit is reached? =
If the API limit is reached, all military codes will not be able to be used until the next month or the plugin is upgraded. The 10 request limit is renewed every 30 days.

= Is the verification accurate? =
Yes. The verification process has direct access to all Active and Honorably Discharged Veterans of the U.S. Armed Forces. Your customers will verify military status by providing their information. All this is handled securely on the veteran verification form.

Our services has been audited and approved by the 
[U.S Department of Veterans Affairs](https://www.va.gov/).

= Does this plugin or service store any personal information? =
No. In order to ensure the privacy of our Military Members and Veterans, this plugin does not store any customer information. Veteran Verify services does not store any of your customers information either. Again, privacy is key.

After customer has verified themselves, they will be issued a temporary key to identify them as verified or not. This plugin will automatically handle this verification process.

= Does this plugin provide additional information about the customer and their military status? =
No. Our service will only provide knowledge if they are an active or honorably discharged member of the U.S. Armed Forces.

= Does this plugin support verifying military members in countries outside the United States? =
No, not at this time but we do plan on adding support for other countries in the future.

= Is this service available for other WordPress plugins? =
At this time, no. With this said, all accounts at [https://veteranverify.app/](https://veteranverify.app/) have direct access to our REST API which allows for custom development and integration into other systems.
You can find our REST API documentation at [https://https://veteranverify.app/docs/api-reference/checking-a-veterans-status/](https://veteranverify.app/docs/api-reference/checking-a-veterans-status/).

= Are customers redirected onsite for verification? =
Yes. When a user requires validation still, they will be redirected to Veteran Verify's secure validation form hosted on our secure server. Once thier they fill in thier information, they will be redirected back to your stores checkout form to continue checkout.
